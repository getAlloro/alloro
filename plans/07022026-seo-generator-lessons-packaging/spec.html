Why

The 2026-07-02 SEO overhaul (six plans, changelogs 0.0.147–0.0.151) taught a set of durable rules — the root causes behind fabricated canonicals, invalid schema types, missing social images, and hallucinated ratings. The code guardrails shipped, but the lessons needed to be captured where the generator lives so future prompt edits and future generation runs don't reintroduce the defects.

What

Package the learnings into the generator mechanism: strengthen the LLM-facing section prompts to close the specific gaps found, and add one comprehensive source-of-truth doc that maps every rule to the layer (prompt / code / renderer) that guarantees it. Documentation + prompt content only — no runtime code or data changes.

Context

Files

  • src/agents/websiteAgents/SeoGeneration.md — base prompt; added "fields set deterministically by the system — do not produce them" (canonical_url, aggregateRating, og_image).
  • SeoGeneration.critical.md — homepage title must never be the bare brand name.
  • SeoGeneration.significant.md — never emit aggregateRating; never hand-author FAQPage; valid business @types only.
  • SeoGeneration.moderate.md — og_image_recommendation is advisory; real og_image is system-resolved.
  • SeoGeneration.LESSONS.md (new) — complete reference: every rule, its motivating failure, and its enforcing layer, plus a quick enforcement-index table. Not loaded into any prompt (documentation only).

Reference analog: the existing SeoGeneration.*.md prompt files (style + loaded-as-text convention).

Constraints

  • Documentation + prompt text only. No TypeScript, no schema, no data writes.
  • SeoGeneration.LESSONS.md must NOT be wired into loadPrompt — it is reference, not prompt tokens.
  • Prompt additions reinforce behavior the code already enforces; they never contradict a code guardrail.

Risk

Level 1. Prompt-content refinements affect only future generation and align with already-shipped code guardrails. The new doc is inert. Blast radius: the shared SEO generation prompt (all orgs), on the next generation run only.
