The Map

The one place to look when in doubt — the same checklist, in the same order, every time. It's the classic buyer journey. We own the three measurable stages in the middle; you work one at a time and never build on a stage that isn't staked.

Awareness→ Research→ Consideration→ Decision→ Loyalty ↑ loyalty (a review) loops back into research + consideration for the next person ↑

How you move

  1. One stage at a time — never skip ahead.
  2. Move on only when the stage you're on is staked.
  3. Go back only one stage, and only if you must.
  4. Never build on a stage that isn't staked. The features come out of a staked map, not before it.

Same steps, same order, every time — whether the case is simple or complicated, you never skip ahead or reorder.

What's on this map — and what's a different machine

On the map: the search path — get found → get considered → get chosen. Someone who already has a need searches, finds you, compares you, reaches out. That is Alloro's lane.

Real, but a different machine — not this map:

  • Professional referrals (a dentist sends a patient to a specialist) — often the biggest channel for a specialist, but we can't control it. We observe it — spot patterns and trends via Reflect (done-with-you). A referral map is possible; its value is unproven, so it's parked, not built.
  • Paid ads (Google Local Services Ads / PPC) — out. It muddies pricing and operations, and a local business can win without it.
