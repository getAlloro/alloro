# Alloro Library (migration from Notion, in progress)

The durable, git-tracked knowledge base for Alloro OS, pushed and Dave-accessible. Replacing Notion as
the source of truth **one deliberate migration at a time** (Corey: "squeeze ALL value out before the
ruthless consolidation; the one unrecoverable mistake is deleting something you needed").

## What's here
- `lattices/` — the three core lattices, migrated from `alloro-brain/library/lattices/` (which are
  themselves mirrors of the Notion databases). Moved here because `alloro-brain` is local-only; this
  repo is pushed and Dave-accessible.
  - `journey-lattice.md` — **CANON.** The Business-Clarity framework (5 stages: Findable / Choosable /
    Bookable / Memorable; the translation + action layers; the undeniable-value tests). Load-bearing;
    the inversion map is built on it.
  - `knowledge-lattice.md` — 74 proven operator heuristics. **status: provisional** (known drift: stale
    MRR, duplicate vintages, ad-era positioning). Notion remains source of truth until reconciled.
  - `sentiment-lattice.md` — the voice / product-surface canon (Watchline, Narrator, No-Tasks, doctor-empathy).

## NOT yet migrated (the follow-on deliberate pass, do WITH Corey)
The broader Notion workspace (~60 canon docs + the Agent Registry, Gold Questions, Harnesses, the
Playbook/Constitution, etc.) is NOT yet mined. That is the "ore, not apparatus" pass: extract the
load-bearing thinking, leave the machinery that made it un-runnable, reconcile drift, THEN Notion is safe
to retire. **Notion is still the source of truth for current canon** (Master Recipe, the moat, MVV, VoC)
until each is migrated and re-established here. Do NOT auto-migrate the apparatus, and do NOT turn Notion
off until the full pass is done and receipted.
