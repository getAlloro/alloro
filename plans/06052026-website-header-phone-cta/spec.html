Website Header Phone CTA

Status
Completed

plans/06052026-website-header-phone-cta
Size
Small

Level 3

Why

Most live website projects have verified phone data, but only TriCity renders a phone CTA in the header. New templates should not inherit TriCity's Spanish helper text as the default.

What

Add a reusable header phone CTA pattern with Call us today as the default helper line. Apply it to the published templates and the existing live headers for Artful Orthodontics, One Endodontics, Garrison Orthodontics, TriCity Endodontics, and Surf City Endodontics.

Context

Relevant Files

  • src/database/migrations/20260421000006_dental_seo_refresh_rev_5.ts - closest reversible template data-migration analog.
  • src/agents/websiteAgents/builder/LayoutGenerator.md - layout agent rules for future generated headers.
  • plans/04212026-no-ticket-dental-seo-template-visual-refresh/preview/template.header.html - local preview artifact for the active dental header template.

Patterns To Follow

  • Use a Knex migration with backup tables and a transaction before changing production-stored HTML.
  • Preserve existing CTA labels, menu shortcodes, logo markup, and mobile-menu mechanics.

Reference file: src/database/migrations/20260421000006_dental_seo_refresh_rev_5.ts - latest template refresh migration pattern.

Constraints

Must

  • Use phone from project_identity.business.phone with primary location fallback.
  • Use Call us today as the default helper line.
  • Skip projects without phone data.

Must Not

  • Do not invent phone numbers.
  • Do not copy TriCity's Spanish helper text into other projects.
  • Do not rewrite unrelated layout HTML.

Out Of Scope

  • Footer cleanup and full layout regeneration are not included.

Risk

Level: 3 - stored production HTML data changes.

Risks Identified

  • Stored header markup varies by project. Mitigation: use guarded insertion patterns and back up target rows before updating.
  • Migration affects production rows after merge to main. Mitigation: reversible backup tables and a narrow host allowlist.

Blast radius: published website-builder templates and project headers for Artful Orthodontics, One Endodontics, Garrison Orthodontics, TriCity Endodontics, and Surf City Endodontics.

Pushback

  • This belongs in the layout/template layer, not page sections. Patching pages would create drift because headers are stored separately and reused across pages.
