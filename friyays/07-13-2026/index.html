Admin and operator notes

Parser assignment now controls formula access

Super admins can set the PMS Parser from the organization overview. “Default (configurable formula)” stores an unset assignment and keeps Formula settings available after columns resolve. Custom parsers such as DentalEMR own their formula, so the formula toggle and sliding mapping drawer stay hidden.

Admin OS keeps more document structure

Internal document imports now preserve supported PDF and Excel images, semantic tables, and readable Markdown structure. Multi-file drops go straight into the Library, and the editor restores the source OS table, image, sticky-toolbar, and safe-Backspace controls.

Internal Alloro tooling Live UI checks waived

Leadgen reporting separates proof from inference

Internal leadgen analytics now keep persisted account links separate from email or domain matches, require audit-backed report activity, and retain historical unaudited rows as clearly marked evidence instead of deleting or presenting them as verified engagement.
