Hi there,

Welcome to Feature Friyay, our weekly roundup of what is new and improving in Alloro.

This week focused on making everyday work faster: cleaner website editing, clearer Local Rankings actions, better form-submission follow-through, and continued progress on Alloro Engage.

  • Faster website editing: routine text, link, image, font-size, and visibility changes can now be handled with direct controls instead of starting from an AI edit request.
  • Inline canvas editing: safe page text can be edited directly on the website canvas, so small copy changes feel more like editing the actual page.
  • Better form submission links: emails with uploaded files now point back to the Website submissions tab for previews and downloads.
  • Clearer Local Rankings actions: rankings refreshes and comparison navigation are easier to find and use.
  • Expanded onboarding: the guided product tour now covers more of the app so new users can get oriented faster.
  • Alloro Engage progress: GBP review replies and local post workflows are moving into a clearer workspace for review queues, drafts, and posting actions.

We will keep tightening these workflows as feedback comes in. The goal is simple: fewer scattered steps, clearer next actions, and a smoother path from insight to execution.

Thanks,
The Alloro Team
