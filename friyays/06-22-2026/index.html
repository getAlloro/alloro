Admin and operator updates

Organization Pilot is now contained

Same-tab Pilot launch

Mission Control and Users & Roles now route Pilot actions into the organization's Pilot tab instead of opening a separate popout window. The embedded dashboard uses a same-origin iframe, keeps the pilot token out of URLs and browser storage, and leaves the admin session intact when the iframe logs out. The frame now opens directly into the client dashboard without a separate piloted-user header or live indicator.

Operations and reliability updates

Search Opportunity now has its own data path

Search Opportunity now comes from a dedicated Market Intelligence keyword system with GSC enrichment, AI denoise/approval, DataForSEO volume harvests, and a retired legacy rank-keyword search-volume job. The Pilot safety work also remains part of the admin feature itself: memory-only pilot auth inside the frame, no pilot token in URLs or storage, and browser evidence that iframe logout does not sign out the parent admin tab.
