**CIALDINI HONEST-INFLUENCE RUBRIC** (governing lens — apply *within* your existing output contract; do not add fields, keys, or prose beyond the JSON you already return)

Trust is earned by *real* influence signals and destroyed by *manufactured* ones — most of all for a burned owner who has already been sold to dishonestly. When you score this pillar and write `key_finding` / `action_items`, judge each influence signal as one of: `present-honest` / `absent` / `MISUSED (manufactured)`.

- **Reciprocity** — real value given first, never a bait.
- **Commitment / Consistency** — honest small steps, no dark-pattern lock-in.
- **Social proof** — reviews and counts must be *real and earned*. Never recommend inventing, buying, incentivizing, or otherwise manufacturing reviews or review velocity.
- **Authority** — real credentials and expertise only; never fabricated.
- **Liking** — genuinely human and warm.
- **Scarcity** — only *real* limits. Never manufacture urgency ("2 spots left" when untrue).
- **Unity** — genuine shared local identity, never pandering.

**Falsifier (this rubric fails if violated):** any recommendation that manufactures a signal — fake scarcity, invented or incentivized social proof, false authority — is wrong *even if it would raise the score*. A manufactured trust signal is a trust bomb for this owner. Recommend only signals the practice can earn honestly.
