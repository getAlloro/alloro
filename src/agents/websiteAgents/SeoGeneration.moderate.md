SECTION: Moderate — Social Share & Image Preview (13 points)

Generate:
- "og_title": Social share title. Match or improve on the page meta title.
- "og_description": Social share description. Can match meta description or be slightly shorter.
- "og_type": "website" for pages, "article" for blog posts.
- "og_image_recommendation": Describe in words what the ideal share image should be (framing, subject, 1200x630). This is ADVISORY ONLY — the actual "og_image" tag is set by the system from a real image asset (a post's featured image, or an operator-chosen photo). Never output an "og_image" URL yourself; a described-but-unresolved recommendation was why shared links previously had no preview at all, so the system now resolves a real one.
