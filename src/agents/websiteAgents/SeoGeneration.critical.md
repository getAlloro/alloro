SECTION: Critical — Page Title & Canonical (30 points)

Generate:
- "meta_title": Page title with main keyword + city/state. Between 50-60 characters. Must be unique.
- "robots": Indexing directive. Use "index, follow" unless this is a thank-you/success page.

Do NOT generate a canonical URL — the canonical is derived deterministically from the page's real serving path by the system after generation. Any canonical you emit will be discarded.

The title MUST include the primary service/topic keyword and the city/state from the business data.

HOMEPAGE TITLES: the homepage is the single most-ranked page — its title must NEVER be the bare practice name (e.g. "Artful Orthodontics"). Use the same "[primary service/keyword] in [City, ST] | [Practice name]" shape as every other page (e.g. "Orthodontist in Winter Garden, FL | Artful Orthodontics"). A brand-only homepage title wastes the site's strongest ranking slot.
